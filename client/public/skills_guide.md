# OpenClaw Skills 综合清单

## 项目背景

本清单为 Vina（OpenClaw 自动化机器人）精心设计，旨在帮助其从"白纸"成长为一个全自动的"数字打工接单机器"。整个工作流遵循**本地轻量级调度 + 云端 API 调用**的架构原则，确保在有限的硬件资源上高效运行。

## 清单结构说明

本文档包含**两份清单**：

1. **第一份：基础 + Web3+AI+自媒体主题 Skills** - 涵盖日常工作所需的通用技能和特定领域的专业工具
2. **第二份：Gemini 讨论方向对应的专项 Skills** - 完整的"接单-执行-交付"闭环工作流，分为四个执行阶段

每条 Skill 包含：**名称**、**来源**、**来源链接**、**功能和作用**、**质量评分**

## 质量评分标准

质量评分基于以下维度进行评估：

- **成熟度**: 是否广泛使用、稳定不崩、文档完善
- **社区活跃度**: 是否有活跃的社区支持、常规更新
- **下载量**: 安装使用数量、下载次数
- **实用性**: 是否解决实际问题、是否有清晰的使用场景

**评分区间**：

- **9.5-10/10**: 高度成熟、广泛使用、文档完善、社区活跃、下载量高（1000+）
- **8.5-9/10**: 成熟稳定、使用者众多、功能完整、下载量中等（500-1000）
- **8/10**: 功能完整、相对稳定、下载量适中（200-500）
- **7.5/10**: 功能基本完整、下载量较低但质量不错（100-200）
